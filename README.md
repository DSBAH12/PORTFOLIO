# Abubakarr Bah - Portfolio

A modern portfolio website showcasing my work, skills, and projects as a Computer Science student at Njala University.

## Technologies Used

- React 18
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons

## Features

- Responsive design
- Technologies showcase
- Project portfolio
- Client testimonials
- Modern UI/UX

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:

   ```bash
   npm install
   ```

3. Start the development server:

   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.
